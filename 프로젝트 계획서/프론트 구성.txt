메인화면
	- 로그인, 회원가입으로 이동
	- 각 게시판

	
로그인 화면
	- 아이디, 비밀번호 (유효성 검사)
	- 회원가입
	- 비밀번호 찾기 (필요에 따라 구현)
	- 카카오 로그인 인증 API (사용 고려)

	
회원가입
	- 아이디
	- 아이디 중복 체크
	- 비밀번호
	- 비밀번호 확인
	- 이름
	- 이메일, 인증번호 전송
	- 주소 (필요에 따라 구현)
	
	
마이 페이지
	- 프로필 사진
	- 로그아웃
	- 정보 수정
	- 문의
	- 내가 쓴 글

	
게시판
	- 글 목록, 글 검색
	- 글 작성, 글 수정, 글 삭제,
	- 댓글 작성, 댓글 수정, 댓글 삭제
	- 카테고리 별 분류
	- 상위 인기 게시글 내림차순 정렬 (필요에 따라 구현)

	
펫 서비스
	- 지도 (산책로, 미용, 호텔, 펜션, 카페, 식당, 장례 장소 등)

	
동물병원
	- 우리동네 동물병원 찾기
	- 반려동물 질병 예측 (설문조사를 통해 증상별 질병 유추)
	- 설문조사 (종, 나이, 성별, 증상 등)
	

동물도감/동물백과사전
	- 종에 따른 분류 : 대분류 (강아지, 고양이 등)
	- 종별 사진과 특징을 출력(ex 강아지 - 말티즈, 푸들, 포메 등)
